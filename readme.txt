MFS campaigns plugin lets you create multiple campaigns on your website and display them randomly on your website. 

== Description ==

This plugin can be used to add campaigns in your website. Admin can create/edit/delete campaigns enable/disable campaigns.

1. It is very easy to add a campaign, just go to admin and locate campaign on the left  menu then click on add new.

2. While adding the campaign you need to provide title , description(if any) , image , campaign link, campaign link text.

3. For each campaign you can have different color bars with different text and different color buttons.

4. After adding your campaign details you just need to hit enter and then it will automatically be added to your site.

5. If you want to disable the campaign then you just need to navigate to Campaigns > settings . Then just uncheck the checkbox and you are done.

== Installation ==

* Download the plugin and extract it.
* Upload the directory '/mfs-campaigns/' to the '/wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Click on 'Campaigns' link to access the admin section.
